## Features

- **Deutsche Inhalte:** Alle Tipps auf Deutsch
- **Smart Filtering:** URL-basierte Filter zum Teilen
- **Modern Design:** Glassmorphism, Glow Effects, Responsive
- **Admin Panel:** Content-Management für neue Tipps

## Tech Stack

| Layer | Technologie |
|-------|-------------|
| Framework | Next.js 15.4 |
| Sprache | TypeScript 5.8 |
| Styling | Tailwind CSS 3.4 |
| Animations | Framer Motion |
| Database | Supabase |
| Analytics | Vercel Analytics |
| Icons | Lucide React |

## Architektur

Atomic Design Methodology:
- **Atoms:** Basis-Elemente (Buttons, Icons)
- **Molecules:** Kombinierte Komponenten (Cards, Forms)
- **Organisms:** Komplexe Sektionen (Header, Filter)
- **Enhanced:** Animierte Varianten
